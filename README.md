# Todo List em React Native

Um todo list pra organizar as tarefas, com opção de buscar tarefas de um servidor ( feito por mim em NodeJS). Os dados estão sendo salvos com async storage.

</p>

* Operações como:
  - Criar
  - Concluir
  - Excluir

## O que foi utilizado

- Typescript
- Styled Components
- Axios
- Async Storage

## Startando projeto
- npm start
